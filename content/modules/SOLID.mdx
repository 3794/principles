# SOLID

로버트 C. 마틴이 정리한 주요 5가지 원칙으로, 첫글자를 따서 만든 단어.

1. [Single Responsibility Principle](/modules/Single-responsibility_principle) (단일 책임 원칙)
2. [Open-Closed Principle](/modules/Open-closed_principle) (개방-폐쇄 원칙)
3. [Liskov Substitution Principle](/modules/Liskov-substitution_principle) (리스코프 치환 원칙)
4. [Interface Segregation Principle](/modules/Interface-segregation_principle) (인터페이스 분리 원칙)
5. [Dependency Inversion Principle](/modules/Dependency-inversion_principle) (의존성 역전 원칙)
