# Design Patterns(디자인 패턴)

- Creational patterns(생성 패턴)
  - Abstract Factory
  - Builder
  - Factory method
  - Prototype
  - Singleton
- Structural patterns(구조 패턴)
  - Adapter
  - Bridge
  - Composite
  - Decorator
  - Facade
  - Flyweight
  - Proxy
- Behavioral patterns(행동 패턴)
  - Chain of Responsibility
  - Command
  - Interpreter
  - Iterator
  - Mediator
  - Memento
  - Observer
  - State
  - Strategy
  - Template Method
  - [Visitor](/concepts/visitor-pattern)
- Concurrency patterns(동시성 패턴)
  - Active Object
  - Lock
