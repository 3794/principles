# Eiffel(programming language)

Eiffel은 1986년, [Bertrand Meyer](/people/Bertrand_Meyer)가 design한 [OOP](/concepts/OOP) 언어이다. Eiffel의 주요 컨셉중 하나는 Design by contract(계약에 의한 설계)이다.

다음은 Hello world 예제이다.

```eiffel
class
    HELLO_WORLD
create
    make
feature
    make
        do
            print ("Hello, world!%N")
        end
end
```
