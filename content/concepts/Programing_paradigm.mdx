# Programing Paradigm(프로그래밍 패러다임)

- Imperative Programming(명령형 프로그래밍)
  - [Object-Oriented Programming(객체 지향 프로그래밍)](/concepts/OOP)
    - Class based(클래스 기반)
    - Prototype based(프로토타입 기반)
  - Procedural Programming(절차형 프로그래밍)
- Declarative Programming(선언형 프로그래밍)
  - [Functional Programming(함수형 프로그래밍)](/concepts/FP)
  - Reactive Programming(반응형 프로그래밍)
- Concurrent Programming(동시성 프로그래밍)
