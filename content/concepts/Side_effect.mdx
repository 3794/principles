# Side effect

operation, function 또는 expression이 side effect를 가진다고 할 때,
이는 해당 operation이 인자의 값을 읽고 호출자에게 값을 반환하는 주된 effect 이외에,
추가적으로 관찰 가능한 effect가 있을 때를 의미한다.
