# Referential transparency(참조 투명성)

프로그램 동작의 변경없이 관련 값을 대체할 수 있다면 해당 expression을 참조 상 투명(referentially transparent)하다고 할 수 있다.

```typescript
const a = 3 + 4;
const b = a * 2;
```

여기서 a는 항상 7이다.
따라서 b를 계산할 때 a 대신 7을 써도 결과는 똑같이 14가 된다.
즉,

- `b = a * 2`
- `b = 7 * 2`

이 두 식은 완전히 같다.
이처럼 어떤 expression을 그 결과값으로 바꿔도 전체 결과가 변하지 않는 것을 말한다.
