# OOP, Object Oriented Programming

[Alan Kay](/people/Alan_Kay)는 객체 지향 프로그래밍에 대해 다음과 같이 말했다.

> 내게 있어 OOP란 오직 메시징, 지역적 보존과 보호, 상태-프로세스의 은닉, 그리고 모든 것에 대한 극단적인 지연 바인딩만을 의미합니다. 이것은 Smalltalk와 LISP에서 가능합니다. 이것이 가능한 다른 시스템들이 있을 수 있지만, 나는 그것들을 알지 못합니다.
