# Functional programming, FP

수학적 의미의 함수에 기반한 프로그래밍 방식 또는 패러다임이다. 함수형 프로그래밍은 공유된 mutable state를 의도적으로 회피한다.
그에 따라 immutable 데이터 구조를 사용한다. 팜수형 프로그래밍은 합성성을 강조한다.[^1]
