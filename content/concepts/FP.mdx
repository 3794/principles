# FP, Functional Programming

functional programing(함수형 프로그래밍)은 프로그램이 함수를 적용하고 조합하여 구성되는 [프로그래밍 패러다임](/concepts/Programing_paradigm)이다.
이는 선언적 프로그래밍 패러다임으로, 함수 정의가 프로그램의 실행 상태를 업데이트하는 명령형 문장들의 시퀀스가 아니라, 값을 다른 값으로 매핑하는 표현식들의 트리 형태를 가진다.
