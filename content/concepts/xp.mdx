# eXtreme Programming, XP (익스트림 프로그래밍)

익스트림 프로그래밍(XP)은 [Kent Beck](/people/Kent_Beck) 등이 제안한 애자일 소프트웨어 개발 방법론 중 하나로, 변화가 잦은 비즈니스 환경에서 소규모 팀이 효율적으로 고품질 소프트웨어를 신속하게 개발할 수 있도록 설계된 방법론이다.
XP는 복잡한 요구사항 변화에 빠르게 대응하는 것을 목표로 하며, 반복적이고 짧은 개발 주기, 단순한 설계, 고객과의 적극적인 소통을 강조한다.
