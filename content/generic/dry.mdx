# Don't Repeat Yourself, DRY(반복되는 코드를 작성하지 마라)

- 코드 중복은 유지보수를 어렵게 만든다
- 버그 발생 가능성을 높인다
- 코드의 일관성을 해친다

```typescript
// 나쁜 예시: 반복되는 코드
function getUserFullName(firstName: string, lastName: string): string {
  return `${firstName} ${lastName}`;
}

function getUserGreeting(firstName: string, lastName: string): string {
  return `Hello, ${firstName} ${lastName}!`;
}

// 좋은 예시: 중복 제거
function getUserGreeting(firstName: string, lastName: string): string {
  const fullName = getUserFullName(firstName, lastName);
  return `Hello, ${fullName}!`;
}
```

반대되는 개념을 [WET](/generic/wet.mdx) 이라고 한다. 중복된 코드를 작성하는 것을 말하며, [AHA](/generic/aha.mdx)와 함께 사용되기도 한다.
