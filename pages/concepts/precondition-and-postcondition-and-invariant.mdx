# Precondition, Postcondition and Invariant (전제조건, 후속조건 그리고 불변식)

function이나 operation이 실행될때 전후나 특정 시점에 만족해야 하는 조건을 말한다.

### Precondition (전제조건)

- 코드나 함수가 실행되기 전에 반드시 만족돼야 하는 조건
- 이 조건이 만족되지 않으면, 코드나 함수의 동작은 보장되지 않음(예: 예외 발생, 잘못된 결과등)

#### Precondition Example

```python
class BankAccount:
    def __init__(self, balance):
        self.balance = balance

    def withdraw(self, amount):
        # precondition: amount must be <= balance
        if amount > self.balance:
            raise ValueError("Insufficient funds")
        self.balance -= amount

```

### Postcondition (후속조건)

- 코드나 함수가 성공적으로 실행된 후에 반드시 만족돼야 하는 조건
- 코드나 함수가 어떤 효과를 가져와야 하는지를 말함

#### Postcondition Example

```python
def withdraw(self, amount):
    if amount > self.balance:
        raise ValueError("Insufficient funds")
    self.balance -= amount
    # postcondition: balance is decreased by amount
```

### Invariant (불변식)

- 프로그램의 실행 중 특정 시점이나 boundary 내에서 항상 참이어야 하는 condition이나 property를 의미
- Invariant는 주로 클래스, 반복문, 또는 알고리즘의 상태를 정의하고 검증하는 데 사용됨

#### Invariant Example

```python
sum = 0
for i in range(n):
    sum += array[i]
    if sum < 0:
        raise ValueError("Sum is negative")
    # Loop Invariant: sum은 항상 0 이상 이어야 함
```
