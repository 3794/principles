# Local Reasoning(지역적 추론)

**Local Reasoning** 은 프로그램의 local만 보고도 그 부분이 올바르게 동작하는지 판단하는 방법이다.
전체 프로그램의 상태나 구조를 몰라도, 특정 함수나 모듈이 자기 스스로의 [전제조건](/concepts/precondition-and-postcondition-and-invariant/#precondition-전제조건)과 [후속조건](/concepts/precondition-and-postcondition-and-invariant/#postcondition-후속조건)만을 이용해서 reasoning 가능한걸 의미한다.

### Example

```typescript
function increment(x: number) {
  return x + 1;
}
```

이 함수는 `x`만 다루기 때문에 `x + 1`만 reasoning 하면 된다.
즉, `x`에만 집중해서 이 함수의 동작을 증명할 수 있기 때문에 local reasoning이다.
