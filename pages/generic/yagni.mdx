# YAGNI

## You Aren't Gonna Need It

그건 필요하지 않을 거야 - 필요하지 않은 기능은 구현하지 마라.

Origin

익스트림 프로그래밍에서 비롯됨

Why

내일 필요할 기능을 위해 오늘 작업하는 것은, 현재 반복(iteration)에서 해야 할 기능에 대한 노력을 낭비하는 것이다.
이는 코드 부풀림(Code bloat)을 야기하며, 소프트웨어가 더 크고 복잡해지게 만든다.

How

실제로 필요할 때만 구현하고, 필요할 것 같다고 예상될 때는 구현하지 마라.
