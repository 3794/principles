## Example API
```java
// java

public class User {
    @NotBlank
    private String username;

    @Schema(nullable = true)
    private String email;
}


public class Post {
    @NotNull
    private Long id;

    @NotNull
    private User user;

    @NotBlank
    private String content;
}

@GetMapping("/posts")
	List<Post> allPost();
```
